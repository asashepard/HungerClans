# HungerClans

Clans plugin for PaperMC. To be used with HungerCore.

Commands:
- /clan OR /c
- /c help
- /c create [clan name]
- /c list
- /c members OR /c members [clan name]
- /c color
- /c banner
- /c [promote|demote|kick|invite] [player]
- /c leave
- /clanwhisper OR /cw OR /cmsg [message...]
- /war
- /war all
- /surrender
- /clanconfig OR /cconfig [key] [value]
